# Pathway Tracker

Off-season training tracker for a 15-year-old footballer (RW/LW/CAM) targeting the pro pathway.

## Block dates
**July 12 – August 31, 2026** (7 weeks)

## Phases
| Weeks | Phase | Focus |
|-------|-------|-------|
| 1–2 | BASE | Build technique and work capacity |
| 3–5 | BUILD | Push intensity, max-effort sprints |
| 6 | PEAK | Highest intensity of the block |
| 7 | TAPER | Cut volume 40%, stay fresh for pre-season |

## Training days
Monday (Max Speed + Agility), Tuesday (Strength), Wednesday (Touches + Ball Mastery),
Friday (Explosive + Position-Specific), Saturday (Speed-Touches Combo under fatigue).

## Tech
Pure HTML/CSS/JS — no framework, no build step. Static site deployed to Netlify.
All data persisted to `localStorage` on the user's device (v1).

## Deploy
Netlify auto-deploys on push to the `main` branch. Publish directory: `.`
